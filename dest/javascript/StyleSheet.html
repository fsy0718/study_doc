Class: StyleSheet

StyleSheet

一个抽象的基础接口，主要描述了文档(HTML/XML)中的一个单独的样式表。当文档是 HTML 时，这个接口描述的可能是<link>标签添加的外部 CSS 文件，也可能是<style>标签添加的一些内部样式。当文档是 XML 时，这个接口描述的是通过<?xml-stylesheet>添加的外部 CSS 文件。
